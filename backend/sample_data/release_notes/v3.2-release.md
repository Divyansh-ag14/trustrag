# AcmeSaaS v3.2 Release Notes

Released: December 1, 2024

## Highlights

This release focuses on platform stability, security improvements, and integration updates.

## New Features

### Custom Fields (Pro and Enterprise)
- Add custom fields to tasks: text, number, date, dropdown, and checkbox types
- Custom fields appear on task cards and are searchable
- Up to 20 custom fields per project (Pro) or 50 per project (Enterprise)

### Project Templates
- Save any project as a template for quick reuse
- Templates preserve workflow stages, custom fields, and default assignees
- Share templates across the workspace

### Enhanced Search
- Full-text search now includes task descriptions, comments, and file names
- Search results are ranked by relevance with highlighted matches
- Filter search results by project, assignee, status, and date range

## Changes

### API Rate Limit Adjustment
- Enterprise plan API rate limits have been adjusted from 10,000 requests/minute to 5,000 requests/minute
- This change was made to improve platform stability during peak usage periods
- Enterprise customers who need higher limits can request a temporary increase through their account manager or by contacting support@acmesaas.com

### Jira Integration OAuth Update
- The Jira integration now requires updated OAuth scopes for improved security
- Existing Jira connections need to be re-authorized: go to Settings > Integrations > Jira > Reconnect
- New scopes include: `read:jira-work`, `write:jira-work`, `read:jira-user` (previously only `read:jira-work` and `write:jira-work`)
- Two-way sync reliability has been improved with this update

### UI Improvements
- Redesigned task detail sidebar with tabbed layout (Details, Activity, Files)
- Improved dark mode contrast ratios for better accessibility
- Faster project switching (reduced load time by 40%)

## Bug Fixes

- Fixed: Tasks with long titles no longer overflow on mobile view
- Fixed: Webhook retries were not respecting the exponential backoff schedule
- Fixed: CSV export was missing tasks created in the last 24 hours
- Fixed: Keyboard shortcut Cmd+N was not working on the Reports page
- Fixed: Two-factor authentication codes were expiring after 15 seconds instead of 30 seconds

## Deprecations

- The v1 API endpoints (`/api/v1/`) are now deprecated. Please migrate to `/api/v2/` by March 1, 2025
- The legacy "Classic" project view has been removed. All projects now use the updated board view
- Email-based task creation (sending emails to create@acmesaas.com) has been discontinued

## Known Issues

- Custom field dropdown values cannot be reordered after creation (fix planned for v3.3)
- Search indexing for newly created tasks may take up to 60 seconds
